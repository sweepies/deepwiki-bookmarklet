# deepwiki-bookmarklet

Simple bookmarklet to open a GitHub repo in DeepWiki

## Installation

1. Create a new bookmark
2. In the **Name** field, enter: "Open in DeepWiki"
3. In the **URL** field, copy and paste the entire contents of [bookmarklet.js](https://github.com/sweepies/deepwiki-bookmarklet/blob/master/bookmarklet.js)
4. Save the bookmark

## Usage

1. Navigate to any GitHub repository
2. Click the bookmark
